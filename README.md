# Li Xiaotong Portfolio

This repository contains a static GitHub Pages personal website for Li Xiaotong.

## Overview

The site presents research interests, reproduced work, startup ideas, and learning paths around robotics, embodied AI, and multimodal systems.

The homepage now supports both English and Chinese, with an in-page language switcher for visitors.

Featured projects:

- DreamVLA
- Tesseract
- VGGT
- OpenVLA

## Local Preview

Open `index.html` directly in a browser. No build step is required.

## GitHub Pages

Use the repository root as the Pages source. GitHub Pages will serve `index.html` as the homepage.
